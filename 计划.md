# 投资最大回撤追踪 — 实施计划

## Context
用户需要一个 Vue Web 应用来追踪投资产品的最大回撤。核心功能：通过东方财富 API 获取每日 K 线数据，记录多次买入操作，计算并可视化从买入起的最大回撤。项目是全新 Vue 3 + Vite 8 脚手架，需从零构建。

## 核心算法：回撤计算

**O(n) 单次遍历**，从首个买入日期开始扫描每日收盘价。

### 指标 1：盈利回撤 (Profit Drawdown) — 主指标

衡量利润回吐程度：赚到的利润跌回去多少。

```
1. 加权平均成本价 = Σ(price × quantity) / Σ(quantity)
2. 遍历每日 K 线:
   - 维护买入后最高价 peak_price
   - peak_profit_rate = (peak_price - cost) / cost     峰值收益率
   - current_profit_rate = (close - cost) / cost        当前收益率
   - 若 peak_profit_rate > 0:
       profit_drawdown = (peak_profit_rate - current_profit_rate) / peak_profit_rate
   - 若 peak_profit_rate ≤ 0 (从未盈利):
       就显示没有盈利
   - 记录最大盈利回撤及其起止日期
```

**举例**: 成本 10 元 → 涨到 20 元 → 回落到 15 元
- peak_profit_rate = (20-10)/10 = 100%
- current_profit_rate = (15-10)/10 = 50%
- 盈利回撤 = (100%-50%)/100% = **50%** (利润回吐一半)

### 指标 2：标准最大回撤 MDD — 辅助指标

衡量价格本身的波动风险：

```
遍历每日 K 线:
  - 维护历史最高价 peak
  - drawdown = (peak - close) / peak
  - 记录最大回撤及起止日期
```

**举例**: 价格 20 → 12 → 18 → 10
- MDD = (20-10) / 20 = **50%** (从最高点跌了一半)

### 输出数据

```typescript
interface DrawdownResult {
  // 盈利回撤 (主指标)
  maxProfitDrawdown: number       // 最大盈利回撤百分比
  profitDrawdownStart: string     // 回撤开始日期 (峰值日)
  profitDrawdownEnd: string       // 回撤结束日期 (谷底日)
  currentProfitDrawdown: number   // 当前盈利回撤

  // 标准 MDD (辅助)
  maxDrawdown: number             // 标准最大回撤百分比
  mddStart: string                // MDD 开始日期
  mddEnd: string                  // MDD 结束日期

  // 基础数据
  costBasis: number               // 加权平均成本
  peakPrice: number               // 买入后最高价
  currentPrice: number            // 当前价格
  totalInvested: number           // 总投入金额
  totalShares: number             // 总份额
  currentValue: number            // 当前市值
  peakProfitRate: number          // 峰值收益率
  currentProfitRate: number       // 当前收益率

}
```
